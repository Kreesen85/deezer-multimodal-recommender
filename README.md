# Deezer Multimodal Recommender Systems

Course project for Recommender Systems.

## Objective
Design and evaluate recommender system approaches for the Deezer DSG17 dataset, focusing on:
- Competition-oriented optimization
- Production-oriented recommendation design

## Repository Structure
- `src/` – Core recommendation models and pipelines
- `notebooks/` – Exploratory analysis and experiments
- `report/` – LaTeX report (CEUR-WS format)
- `data/` – Dataset description only (no data files)
