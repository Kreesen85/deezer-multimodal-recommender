# Notebooks

This directory contains exploratory analysis and experiments for the Deezer Sequential Skip Prediction project.

## Files

### `eda_analysis.py`
Comprehensive exploratory data analysis script covering:
- Data overview and quality checks
- Target variable distribution and class imbalance
- User demographics analysis (age, gender)
- Track and content features (duration, genres, artists)
- Temporal patterns (time of day, day of week)
- Platform and context analysis
- Feature correlations
- Summary statistics and insights

**Run it:**
```bash
cd notebooks
python eda_analysis.py
```

**Outputs:**
- Multiple visualization PNG files
- `eda_summary.txt` with key findings

### `exploration.ipynb`
Interactive Jupyter notebook for exploratory analysis (create as needed)

### `experiments.ipynb`
Experimental model development notebook (create as needed)

## Setup

Make sure you have installed the required packages:
```bash
pip install -r ../requirements.txt
```
